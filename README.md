# 3D_Beginne-Roll-a-Ball_Game
1. Візуал та Аудіо
Зміненно середовище.
Audio System.
VFX: Particle System.
2. UI та Data Persistence
UI Flow: Main Menu та End Game Panel (вивід статистики, рахунку, таймера, лоад сцен).
Data Persistence: Збереження налаштувань через PlayerPrefs. UI-компоненти автоматично синхронізують стейти між сценами.
Levels Game: Реалізованно два типи рівня.
3. Custom Mechanics 
Runtime Skin Swap: Динамічна зміна матеріалів гравця.
Jump Logic: Механіка стрибка.
Динамічне створення бонусів та Супер бонусів.
Time Limit: Обмеження в часі для режиму Time Attack.
