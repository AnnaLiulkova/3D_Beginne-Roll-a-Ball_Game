# 3D_Beginne-Roll-a-Ball_Game
### Visuals/Audio
* **Оновлене середовище:** Покращено візуальне оформлення ігрових локацій (модельки скачені з Asset Store).
* **Audio System:** Background Music, звуки взаємодії з UI (натискання кнопок), ефекти підбирання звичайних і супербонусів, а також мелодії перемоги та програшу.
* **VFX:** Particle System для візуалізації програшу гравця. Також є і для ворога.

### UI
* **UI Flow:** MainMenu з керуванням панелями налаштувань, меню паузи під час гри та EndGamePanel (вивід фінальної статистики: рахунок, таймер та можливість перейпроти гру).
* **Data Persistence:** Налаштування гри (гучність, скіни, кількість ворогів) та стани UI-компонентів автоматично зберігаються та глобально синхронізуються між усіма сценами.

### GameModes
Реалізовано два типи ігрових рівнів:
* **Level 1 (Time Limit):** Режим на збір очок із обмеженням у часі.
* **Level 2 (Survival):** Режим на виживання, де гра триває доти, доки гравця не спіймають вороги. Під час гри з'являються "Супер бонуси"

### Custom Mechanics
* **Jump Logic:** Реалізована механіка стрибка гравця.
* **Динамічний спавн:** Створення звичайних бонусів та Супер Бонусів (з'являються кожні 20 секунд та додають +5 очків) в ігровому режимі Level 2.
* **Skin Swap:** Зміна зовнішності гравця.
* **Enemy count:** Можливість гравцеві самостійно визначати кількість ворогів.
